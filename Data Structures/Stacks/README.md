# Stacks

A sequential data collection where we can only add and remove elements at the top.
Last in first out structure (LIFO).

| Operation | Pseudocode   |
|-----------|--------------|
| top       | TOP[S]      |
| pop!      | POP[S]   |
| push[p]   | PUSH[p, S]|
| empty?    | EMPTY[S]     |

Construct new empty stack

`new Stack S`
